# Frontend Structure

The React client will be organized around reusable UI and route-level pages.

## Planned folders

- `src/api` request helpers and fetch wrappers
- `src/assets` static assets
- `src/components` reusable UI components
- `src/context` auth and app-level providers
- `src/hooks` custom React hooks
- `src/layouts` route layouts
- `src/pages` public, user, and admin pages
- `src/routes` route guards and route config helpers
- `src/styles` Tailwind and shared CSS
- `src/utils` formatting and validation helpers
